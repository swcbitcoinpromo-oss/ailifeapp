# Coach avatars — ORIGINAL characters only

Drop one image per coach here, used by the Runway briefing-video pipeline:

- `atlas.png` — the planner (calm, sharp)
- `forge.png` — the trainer (intense, fiery)
- `sage.png` — the nutritionist (measured, precise)

Rules (build brief §2 — the one ground rule):

- Original illustrated or AI-generated characters that you own.
- NOT real people. No celebrity faces, no lookalikes of named public figures,
  no cloned voices. This keeps the app App-Store-safe and lawsuit-proof.

Square PNGs around 1024×1024 work well as Runway image-to-video inputs.
